## Design Decisions (locked in)

### Document Styling
- **Font:** Helvetica
- **Colors:** Black (#1A1A1A), Grey (#666666), Light Grey (#AAAAAA), Line Grey (#CCCCCC)
- **Dividers:** Grey horizontal lines between sections
- **Heading hierarchy:** H1 (name) → H2 (professional title) → H3 (section headings + employer entries) → H4 (individual projects)
- **Minimal formatting** — no accent colors, no heavy styling
- **Bullet format:** 3 bullets per project, structured as:
  1. **Context** — what the project was (format, frequency, audience, duration)
  2. **Ownership** — what Hisham specifically did
  3. **Scale/Outcome** — numbers, delivery record, impact

### Resume Structure
- No italic context line under project titles — context is folded into bullet 1
- No "Additional Experience" section (music, ROTA, Northwestern dropped)
- IBM Capstone gets its own entry (separate from Education)
- Credential verification link included in Education section
